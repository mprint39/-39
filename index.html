📋 Инструкция по использованию калькулятора MPrint_39

🎯 Основные вкладки

  • 1. Расчет - для стандартной продукции
    Выбирайте тип продукции, формат, цветность, плотность и материал
    Указывайте нужное количество
    Система автоматически рассчитает стоимость
  • 2. Нестандарт - для индивидуальных размеров
    Если не нашли нужный размер в основных вариантах
    Укажите точные размеры в миллиметрах
    Выберите материал и параметры печати
    ⚡ Автоматически добавлен промежуток 3мм между изделиями для правильной нарезки
    Получите расчет для вашего индивидуального заказа
  • 3. Наклейки - расчет количества на листе
    Узнайте, сколько наклеек поместится на листе
    Выберите тип листа (SRA3 или SRA4)
    Укажите размеры ваших наклеек
    ⚡ Автоматически добавлен промежуток 3мм между наклейками для правильной вырубки
    После расчета вы сможете определить необходимое количество листов
  • 4. Контакты - связь с типографией
    Наши реквизиты и контактная информация
    Быстрая связь через Telegram

🎨 Расшифровка цветности

  • 1+0 - Односторонняя черно-белая печать
  • 1+1 - Двухсторонняя черно-белая печать
  • 4+0 - Односторонняя цветная печать
  • 4+4 - Двухсторонняя цветная печать

📏 Важно о промежутках

  • В расчетах нестандартных изделий и наклеек автоматически учитывается технологический промежуток 3мм
  • Это необходимо для качественной нарезки и предотвращения повреждения краев
  • Фактический размер готового изделия будет соответствовать указанным вами размерам

✂️ Дополнительные услуги

Для следующии услуг уточняйте стоимость у оператора:

  • Контурная резка - точная вырубка по контуру
  • Ламинация - защитное покрытие (односторонняя/двухсторонняя)
  • Биговка - создание линий сгиба
  • Брошюровка - книжный переплет и степлер
  • Нарезка - подготовка бумаги в нужный формат
  • Дополнительная обработка - индивидуальные решения

💡 Советы по использованию

  • Всегда проверяйте выбранные параметры перед добавлением в заказ
  • Для нестандартных размеров используйте соответствующую вкладку
  • Если сомневаетесь в выборе материала - проконсультируйтесь с оператором
  • Несколько позиций можно добавлять в корзину и заказывать одной кнопкой
  • Учитывайте, что промежутки 3мм уже включены в расчеты
